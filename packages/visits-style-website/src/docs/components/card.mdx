## Imports
```javascript
import { Card } from 'visits-style';
```

## Basic usage

```jsx live
<Col size={4} offset={4} auto>
  <Card title="Card title">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    Aperiam deleniti fugit incidunt, iste, itaque minima neque pariatur perferendis
    sed suscipit velit vitae voluptatem. A consequuntur, deserunt eaque error nulla temporibus!
  </Card>
</Col>
```
