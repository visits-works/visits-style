AppBar

nametyperequireddefaultdescription
colorstringfalseなしbackground色
brandanyfalseなしロゴのイメージ、プロジェクト名など
fixedbooleanfalseなし定義された位置を固定にする
stickybooleanfalseなし(IE11不可)画面がスクロールされても上で貼り付けいるようにする
fluidbooleanfalseなし中央並びから自動幅で表示します
backdropbooleanfalseなし背景がblurされます（safari専用、他は透明度）
align"left" | "right"falseなしchildrenに定義するElementの並び順を指定します。未定義は自動並び

import { AppBar } from 'visits-style';
