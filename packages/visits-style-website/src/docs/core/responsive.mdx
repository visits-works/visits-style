サイズの定義は [theme](/core/theme#variables)の値を参照してください。

## Before
まずは`theme`の定義で、`responsive`が`true`なのか確認してください。

デフォルトは`true`です。

## 定義方法

```javascript
import { css } from 'styled-components';
import { mediaMobile } from 'visits-style';
css`
  display: none;

  ${mediaMobile} {
    display: block;
  }
`;
```
ようにすると、


```css
display: none;
@media and screen (max-width: 576px) {
  display: block;
}
```
のように適用されます。
