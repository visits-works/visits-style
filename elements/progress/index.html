Progress

進捗バー

nametyperequireddefaultdescription
valuenumbertrueなし現状の進捗
maxnumbertrueなし進捗の最大値
sizeSizeTypefalseなしバーのサイズ
heightstringfalseなしsizeを使わないときの縦幅を指定する
colorstringfalseなしバーの色

Import

import { Progress } from 'visits-style';

Props

Basic usage

<Progress value={20} max={100} />

Sizes & custom height

sizeとheightは一緒に使えません。

sizeが優先に適用されます。




<>
  <Progress value={20} max={100} height="0.25rem" />
  <br />
  <Progress value={20} max={100} size="small" />
  <br />
  <Progress value={20} max={100} size="medium" />
  <br />
  <Progress value={20} max={100} size="large" />
</>
