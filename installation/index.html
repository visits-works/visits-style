    Installing Visits-style

    reactとreact-domとstyled-componentをPeerDependenciesとして使っています。

    npm install --save github:visits-works/visits-style#master styled-components react react-dom

    yarnを使う場合は、

    yarn add github:visits-works/visits-style#master styled-components react react-dom
