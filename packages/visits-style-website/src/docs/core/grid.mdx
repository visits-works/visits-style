## Import

```javascript
import { Col, Container, Row } from 'visits-style';
```


## Basic usage

```jsx live
<Container>
  <Row>
    <Col><Box color="link">1</Box></Col>
    <Col><Box color="link">1</Box></Col>
    <Col><Box color="link">1</Box></Col>
    <Col><Box color="link">1</Box></Col>
  </Row>
</Container>
```

## Offset

```jsx live
<Container>
  <Row>
    <Col offset={2} size={4}><Box color="link">4-2</Box></Col>
    <Col offset={1} size={2}><Box color="link">2-1</Box></Col>
    <Col offset={1} size={2}><Box color="link">2-1</Box></Col>
  </Row>
</Container>
```


## Break

```jsx live
<Container>
  <Row multiline>
    <Col><Box color="link">Lorem</Box></Col>
    <Col><Box color="link">ipsum</Box></Col>
    <Break />
    <Col><Box color="link">dolor</Box></Col>
    <Col><Box color="link">sit</Box></Col>
  </Row>
</Container>
```
