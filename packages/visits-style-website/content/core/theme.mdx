import { Table, Tag } from '@components';
import theme from '@theme/default';
import Cube from '_components/Cube';

プロジェクト事の設定値です。

## example

基本的に内臓されているコンポーネントは以下の設定値として作られているのですが、

以下のようにカスタムとして値を使えます。

```javascript
css`
  color: ${({ theme }) => theme.primary};
`;
```
