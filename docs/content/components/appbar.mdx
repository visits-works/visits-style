## Props

## Basic usage

```jsx live
<AppBar color="light" brand={(<Logo />)} to="#">
  <ul>
    <li><a href="#">link1</a></li>
    <li><a href="#">link2</a></li>
    <li><a href="#">link3</a></li>
  </ul>
  <div>
    <Button color="warning" outline>Link</Button>
  </div>
</AppBar>
```
