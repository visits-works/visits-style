## Basic Usage

```jsx live
<Hero>
  <h1>Title</h1>
  <h2>Subtitle</h2>
</Hero>
```

## Sizes

default size is `small`

```jsx live
<>
  <Hero size="medium" color="primary">
    <h1>Medium</h1>
    <h2>medium subtitle</h2>
  </Hero>
  <br />
  <Hero size="large" color="primary">
    <h1>Large</h1>
    <h2>large subtitle</h2>
  </Hero>
  <br />
  <Hero size="full" color="primary">
    <h1>FullHeight</h1>
    <h2>full subtitle</h2>
  </Hero>
</>
```

## Centered Text
```jsx live
<Hero color="light" center>
  <h1>Center Title</h1>
  <h2>center subtitle</h2>
</Hero>
```

## with Appbar
```jsx live
<Hero
  color="dark"
  header={(
    <AppBar brand={(<Logo />)} to="#">
      <ul>
        <li><a href="#">link1</a></li>
        <li><a href="#">link2</a></li>
        <li><a href="#">link3</a></li>
      </ul>
    </AppBar>
  )}
  center
>
  <h1>Title</h1>
  <h2>subtitle</h2>
  <p>some text here</p>
</Hero>
```
